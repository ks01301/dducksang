- 완료 후 컴퓨터 재부팅 (권장)

🔐 키움증권 계좌로 로그인

**끝! 이제 자동매매를 시작하세요!** 🎉

---

## 📋 초기 설정 (처음 1회만)

### 매매 전략 설정

1. **설정** 탭 클릭
2. 다음 값 입력:
   - K값: `0.5` (권장)
   - 손절율: `2` (2% 손실 시 자동 매도)
   - 익절율: `5` (5% 수익 시 자동 매도)
3. **"설정 저장"** 버튼 클릭

### 스캔 프로필 선택

- **전고점 돌파**: 공격적 (변동성 큰 종목)
- **정배열 & 골든크로스**: 안정적 (추세 추종)
- **볼린저 밴드**: 변동성 기반

### 자산 관리

1. **자산관리** 탭 클릭
2. **"운용 자금 초기화"** 클릭
3. 투자할 금액 입력 (예: 1,000,000원)
4. **"확인"** 클릭

---

## ⚠️ 중요 안전 수칙

### 🔴 필수 체크

- [ ] **모의투자로 먼저 테스트** (3~5일)
- [ ] **소액으로 시작** (100만원 이하)
- [ ] **손절율 반드시 설정** (권장: 2%)
- [ ] **정기적으로 로그 확인**

### 💰 리스크 관리

```
추천 시작 금액: 50만원 ~ 100만원
손절율: 2% (필수)
익절율: 5% (권장)
종목당 한도: 20만원 이하
```
