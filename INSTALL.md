# 🚀 DDuckSang 자동매매 프로그램 설치 가이드

## 📋 시스템 요구사항

- **운영체제**: Windows 7/10/11 (32비트 또는 64비트)
- **키움증권 계좌**: 필수
- **인터넷 연결**: 필수

---

## 📥 설치 절차

### 1. 키움 OpenAPI+ 설치

1. [키움증권 OpenAPI+ 다운로드](https://www.kiwoom.com/nkw.templateFrameSet.do?m=m1408000000)
2. 다운로드한 설치 파일 실행
3. 기본 설정으로 설치 완료
4. PC 재부팅 (권장)

### 2. Visual C++ Redistributable 설치 (필요 시)

일부 환경에서는 Microsoft Visual C++ 재배포 패키지 필요:

- [Visual C++ 2015-2022 Redistributable (x86)](https://aka.ms/vs/17/release/vc_redist.x86.exe)

### 3. 프로그램 실행

1. `DDuckSang_AutoTrading.exe` 더블클릭
2. 키움증권 로그인 화면에서 계좌 로그인
3. 매매 설정 후 자동매매 시작!

---

## ⚙️ 초기 설정

### 1. 매매 전략 설정

- **K값**: 0.4 ~ 0.6 권장 (기본: 0.5)
- **손절율**: 2% 권장 (양수 입력)
- **익절율**: 5% 권장 (양수 입력)

### 2. 스캔 프로필 선택

- **전고점 돌파**: 공격적 매매
- **정배열 & 골든크로스**: 안정적 매매
- **볼린저 밴드**: 변동성 기반
- **사용자 정의**: 맞춤 설정

### 3. 자산 관리

- 운용 자금 설정
- 종목당 매수 한도 설정

## 🔒 보안 주의사항

1. **공인인증서/비밀번호**: 안전하게 보관
2. **자동로그인**: 비활성화 권장
3. **PC 보안**: 백신 프로그램 설치

## 📞 문제 해결

### "OCX 컨트롤을 등록할 수 없습니다"

→ 키움 OpenAPI+ 재설치

### "로그인 실패"

→ 키움증권 홈페이지에서 로그인 테스트

### "프로그램이 실행되지 않음"

→ Visual C++ Redistributable 설치

---

## 💾 데이터 백업

### 자동 생성 파일

- `trading.db`: 거래 내역 데이터베이스
- `strategy_config_*.json`: 전략 설정
- `logs/`: 매매 로그

### 백업 방법

1. 설정 탭 → "📤 백업 (Export)" 클릭
2. 백업 폴더 선택
3. 다른 PC에서 복원 시: "📥 복원 (Import)" 사용

---

## 🎯 추천 운용 방식

1. **모의투자로 시작**: 3~5일 테스트
2. **소액으로 시작**: 100만원 이하
3. **점진적 확대**: 안정성 확인 후
4. **지속 모니터링**: 정기적으로 로그 확인
