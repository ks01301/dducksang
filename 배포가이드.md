## 🚀 배포 방법

### 방법 1: USB 배포

```
1. release 폴더 전체를 USB에 복사
2. 다른 PC에 붙여넣기
3. 시작하기.md 따라하기
```

### 방법 2: 압축 파일 배포

```
1. release 폴더를 ZIP으로 압축
2. 이메일/클라우드로 전송
3. 압축 풀고 시작하기.md 확인
```

### 방법 3: 네트워크 공유

```
1. release 폴더 공유 설정
2. 다른 PC에서 접근
3. 실행
```
